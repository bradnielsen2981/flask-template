JSON Example

JSON is a method of sending complex data through text encoding and also avoiding a page refresh.

I have a created a helper function which can be used as follows: JSONrequest("/url", "POST",name_of_callback_function, {var1:"value1",var2:"value2"...})

The callback function receives a (result) object that contains response data.

A simpler call can be made: JSONrequest("/url", "POST") if you do not wish to send or receive data.
